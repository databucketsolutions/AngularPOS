# Avocado
an Angular Point of Sale system


## Tech
* Angular
* NodeJS
* ExpressJS
* MongoDB
* Mongoose

## Installation

### Prerequisites
* NodeJS
* MongoDB

### Getting Started
Install dependencies and start development server

```sh
$ git clone https://github.com/ArashDai/Avocado.git

$ npm install

$ npm start

```

Navigate to localhost:3000
