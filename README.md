# DataBucket Solutions
an Angular Point of Sale system


## Tech
* Angular
* NodeJS
* ExpressJS
* MongoDB
* Mongoose

## Installation

### Prerequisites
* NodeJS
* MongoDB

### Getting Started
Install dependencies and start development server

```sh
$ https://github.com/databucketsolutions/AngularPOS.git

$ npm install

$ npm start

```

Navigate to localhost:3000
